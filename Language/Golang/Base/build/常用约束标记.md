# 常用约束标记



### 操作系统标记

```bash
darwin    // macOS
linux     // Linux
windows   // Windows
freebsd   // FreeBSD
netbsd    // NetBSD
openbsd   // OpenBSD
plan9     // Plan9
```



### 架构标记

```
386      // 32位x86
amd64    // 64位x86
arm      // ARM
arm64    // ARM64
mips     // MIPS
mips64   // MIPS64
ppc64    // PowerPC64
ppc64le  // PowerPC64小端
riscv64  // RISC-V64
s390x    // IBM zSystems
wasm     // WebAssembly
```



### 其他标记

```
cgo      // 启用CGO
race     // 竞态检测
debug    // 调试模式
test     // 测试环境
```
